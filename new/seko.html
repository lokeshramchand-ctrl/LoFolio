SEKO — Instant Commerce for Your Community

Seko is a hyperlocal marketplace that connects community sellers and local brands with customers in real time. Inventory syncing, routed logistics, and a refined shopping experience — all built end-to-end.

Role: Full-Stack Engineer
Tech: Next.js · Node · PostgreSQL · Redis
Year: 2024

Project Overview

SEKO aims to solve the biggest gap in Indian hyperlocal commerce: local sellers struggle with visibility and customers struggle with reliability. We built a complete system that connects small businesses, provides real-time inventory, handles logistics, and keeps the experience ultra-fast and simple.

From API architecture to seller onboarding, real-time delivery tracking, authentication, and UI design — the entire system was engineered to scale smoothly across entire apartment complexes.

Engineering Approach

We architected SEKO with a modular micro-services approach — separating auth, orders, inventory, geolocation, and deliveries. Redis caching + connection pooling allowed under-100ms responses even during surge hours.

A custom admin panel was built for sellers to track orders, manage stock, generate automatic invoices and handle real-time order confirmations.
