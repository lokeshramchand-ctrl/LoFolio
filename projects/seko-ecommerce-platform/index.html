Project — Seko HyperLocal Marketplace

Seko.
Community Commerce Unified.

A unified digital hyperlocal marketplace ecosystem for agricultural commerce, built with Flutter for cross-platform mobile access, Django for a scalable REST backend, and PostgreSQL for transactional consistency. Features secure authentication, inventory management, AI-powered product classification, cart operations, and optional recommendation systems powering community supermarkets and local farmers.
