VELAR

Velar is a personal finance system built to eliminate manual expense tracking. Convert unstructured email data into structured financial intelligence with automated parsing and clean visual analytics.

Core Arsenal

Flutter Mobile

A cross-platform application designed for clarity, providing dashboards and spending insights that update in real time.

Node.js Parser

A high-accuracy email parser that extracts financial data from inconsistent bank emails.

RabbitMQ

Decouples heavy parsing from the UI, ensuring the app remains responsive during ingestion.

Docker & CI/CD

Automated testing and deployment pipelines for consistent multi-service environments.

Technical Deep Dive

Regex Parsing Engine

Velar uses a custom regex engine identifying transactional patterns with high accuracy from noisy email bodies.

Self-Learning Categorizer
