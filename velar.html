VELAR

Velar is a personal finance system built to eliminate manual expense tracking. Most users receive dozens of transactional emails every month but rarely organize them. Velar bridges this gap by converting unstructured email data into structured financial intelligence with automated parsing, instant classification, and clear visual analytics.

Core Arsenal

Flutter Mobile

A cross-platform application designed for clarity, providing dashboards, category breakdowns, and spending insights that update in real time.

Node.js Parser

A high-accuracy email parser that extracts merchant, amount, date, and category from inconsistent and unstructured bank emails using an advanced Regex interpretation pipeline.

RabbitMQ

Decouples heavy parsing and analytics from the UI. Incoming emails are processed in background workers, ensuring the app remains smooth and responsive.

Docker & CI/CD

Every service runs inside a container with automated testing and deployment pipelines, enabling consistent environments and rapid releases.

System Design

Async Ingestion.

Financial parsing is compute-heavy. Instead of blocking the UI, Velar routes every incoming email through an asynchronous ingestion pipeline. The Email Hook forwards raw content to RabbitMQ, where worker nodes clean, categorize, and store the transaction without interrupting user activity.

Technical Deep Dive

01

Regex Parsing Engine

Bank emails vary widely in format and structure. Velar uses a custom regex engine capable of identifying transactional patterns with high accuracy, extracting merchant names, dates, amounts, and references even from noisy email bodies.

02

Self-Learning Categorizer

Velar improves over time. When users correct a miscategorized transaction, the system adjusts merchant-category weightings. This lightweight adaptive model avoids heavy ML pipelines while still delivering improving accuracy.

03

Automated Deployment

The backend is fully containerized with Docker, and a CI/CD pipeline automates testing and deployment. Every commit triggers validation, image building, and registry updates, reducing deployment effort by 70 percent.
